## Context
Full design doc (architecture, API, schema, next steps): `~/.gstack/projects/capacitor-updater-server/nikitaaplin-main-design-20260325-192709.md`
Stack: Next.js 16 + React 19 + Drizzle ORM + Postgres + MinIO + shadcn/TW, all in Docker Compose.
Core: Self-hosted server implementing `@capgo/capacitor-updater` protocol for OTA bundle updates.

## Vision

### 10x Check
The 10x version is a full open-source Capgo replacement: channels with percentage rollouts, auto-rollback on crash rate spikes, bundle signing/encryption, CLI tool (`npx cap-updater deploy`), GitHub Actions integration, multi-app support from a single server, and a community plugin marketplace. That's an ocean for v1 — but the API-first architecture of the current plan lays the foundation for every one of these features.
